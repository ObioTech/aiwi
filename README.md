# ObiBrain — Local-first Project Brain for AI Agents

ObiBrain is a local-first Project Brain for AI agents.

ObiBrain helps AI agents deeply understand software projects through documents, code context, human-approved memory, and operational evidence.

**Core Philosophy:**
- **Local-first & SQLite-first:** Keeps data close to your project without depending on external cloud databases for core indexing.
- **MCP-native:** Built natively to expose Model Context Protocol (MCP) tools for any compatible AI agent.
- **Project Brain, not a Workflow Suite:** ObiBrain acts as the unified intelligence layer for your project, exposing context and memories that any workflow or agent profile can leverage.

For more details, concept docs, and public demos, visit our [GitHub Pages Documentation](https://obiotech.github.io/aiwi/).
