# AIWI — AI Workflow Infrastructure
**A local-first Project Brain for AI agents**

## What is AIWI?
AIWI helps AI agents understand your software project. By indexing documents, code context, human-approved memory, and operational evidence locally, AIWI provides a robust "Project Brain" that AI assistants can seamlessly query via Model Context Protocol (MCP).

## Core Ideas
- **Local-first & SQLite-first**: Fast, local, and portable project indexes.
- **MCP-native**: Standardized toolsets for AI agents to explore the project.
- **Read-only Linking**: Connect multiple Project Brains together securely.
- **Human-approved Memory**: Governance over architectural decisions, conventions, and bug patterns.
